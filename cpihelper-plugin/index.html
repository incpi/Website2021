CPI Helper extension for chrome : Trace Step Modifier
Thisplugin lets user/Develper use trace step modifier in ntergration flows.
Carefull while using this because it might be ended up in browser freeze.
Issue:
Sometimes, Using CPI helper doesn't take all steps in Inline debug mode. This plugin will help you with this issue to have more flexible control.
Configuration (Without Plugin):
you want to change this configuration for all the flows. then go to extention page where you configure theme & tab name,etc.
Set as any number
Chnages applied to all the flows. 300 (default) - Safest option
Set as 0
This willremove the limit at all. but for large flows. this will freeze up whole browser.
Set as null or leave it blank
This will use default value
Configuration (With Plugin):
This will allow to change globalvariable for each Integration flow. This will overwrite global variable which will be applicable to tentent.
